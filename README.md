# mobi-style-guide

### mobi-path

대상자
```
취업 상태 이전의 모든 mobi
```

조건
```
주당 30시간 이상 학습, 데일리 스크럼 작성, 과제 수행
```

내용
```
약 6개월간 기초 학습 시행, 페어프로그래밍과 동료 학습을 통해 조별로 진행
모비 패스의 동료 학습 조는 모비 커뮤니티 조와 동일
```

탈락
```
아래와 같은 사유의 인원은 모비 패스에서 탈락됩니다

1. 데일리 스크럼 누락자 (경고 1회)
2. 과제 수행 불가
3. 모비 패스 학습 과정에 성실히 임하지 않는 자
```
